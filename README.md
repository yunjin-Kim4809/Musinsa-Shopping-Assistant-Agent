# 🛍️ 무신사 쇼핑 도움 에이전트

무신사(Musinsa) 쇼핑을 돕는 CLI 에이전트입니다.  
**Tavily API**로 웹 검색을 하고, 일부 기능에서는 **OpenAI API**(GPT-4o)를 사용합니다.

---

## ✨ 주요 기능

| 메뉴 | 기능 | 설명 |
|------|------|------|
| **1** | 제품 가격·평점 비교 | 두 제품의 가격, 할인, 평점, 스펙을 비교하고 가성비 순으로 추천 |
| **2** | 취향 기반 제품 추천 | 한 줄 키워드(스타일, 예산, 브랜드)로 무신사 위주 추천 3개 |
| **3** | 리뷰 장단점 요약 | 제품명 입력 시 무신사 리뷰를 수집해 주제별 장단점 요약 |
| **4** | 취향 기반 추천 (인터랙티브) | 메뉴로 카테고리·스타일·브랜드 선택 후 GPT-4o로 관련성 점수 부여해 상위 5개 추천 |

---

## 📋 요구 사항

- **Python 3.8+**
- **TAVILY_API_KEY** (필수) — [Tavily](https://tavily.com)에서 발급
- **OPENAI_API_KEY** (선택) — 4번 메뉴 사용 시 필요

---

## 🚀 설치 및 실행

### 1. 저장소 클론 및 의존성 설치

```bash
cd hateslop_agent_hackathon
pip install -r requirements.txt
```

### 2. API 키 설정

프로젝트 폴더에 **`.env`** 파일을 만들고 아래 형식으로 입력합니다.

```env
TAVILY_API_KEY=여기에_타빌리_API_키
OPENAI_API_KEY=여기에_오픈AI_API_키
```

- 1·2·3번만 쓸 경우 `TAVILY_API_KEY`만 있어도 됩니다.
- 4번(취향 기반 인터랙티브 추천)을 쓰려면 `OPENAI_API_KEY`도 필요합니다.

> ⚠️ `.env` 파일은 git에 올리지 마세요. (민감 정보 포함)

### 3. 실행

```bash
python main.py
```

실행 후 메뉴에서 원하는 번호를 입력하면 됩니다.

## 📖 사용 예시

### 1번 — 제품 비교

- 입력 예: `나이키 에어 포스 1 07 W - 화이트와 푸마 터프패디드 FS 코듀로이`
- 또는: `A 코트 vs B 코트`

### 2번 — 취향 키워드 추천

- 입력 예: `미니멀리즘, 30만원대, 아르켓 느낌`
- 또는: `스트릿 스타일, 20만원, 나이키`

### 3번 — 리뷰 요약

- 입력 예: `나이키 에어맥스`, `무신사 코트`

### 4번 — 인터랙티브 추천

- 카테고리(상의/하의/아우터 등) → 스타일(미니멀/캠퍼스/스트릿 등) → 브랜드·가격대 등 선택
- 필요 시 자연어로 추가 설명 입력 가능
